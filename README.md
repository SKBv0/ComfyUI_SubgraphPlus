# ComfyUI SubgraphPlus

Opens subgraphs in a draggable, resizable popup window so you can view and edit them without leaving the main workflow.
<img width="1272" height="664" alt="subg" src="https://github.com/user-attachments/assets/9cb59ac0-0caf-4308-9b34-157ce1e0f3fc" />

## Installation

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/SKBv0/Comfyui_subgraphplus.git
```

## Usage

1. Right-click a subgraph node.
2. Choose **Show Subgraph**.
3. You can open multiple subgraphs at once. Right-click the main canvas → **Close All Subgraphs** to close them all.

Changes made in the popup are reflected in the main workflow, and vice versa.
